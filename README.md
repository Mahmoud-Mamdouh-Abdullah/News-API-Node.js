# News-API
News API for create, read, update and delete news

## Using :
- Node.js.
- Express.js.
- TypeScript.
- MongoDb
- Service Repository Controller pattern.

## The Project Covers : 
- add user.
- login and generate a token.
- get all users.
- get all news.
- add news.
- get news by id.
- get news by title or content.
- delete news by id.
- edit news by id.
- add socket.io to the server and create a simple chat app.
